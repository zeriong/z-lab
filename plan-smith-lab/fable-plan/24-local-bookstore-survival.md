# 동네 서점의 생존 전략 1년 플랜
- 추론 프레임: 관찰-데이터 우선
- 한 줄 요약: 관찰 항목마다 "이 데이터가 갈라놓을 결정"을 사전에 명시한다 — 결정과 연결되지 않은 관찰은 수집하지 않는다. 전략은 관찰의 분기(if-then)로만 존재하고, 1년 플랜의 실체는 "관찰→개입→재관찰"의 분기 루프다.

## 프레임이 요구하는 시작점 — 관찰 설계 (결정 연결 원칙)

전략 아이디어(북토크! 카페! SNS!)를 봉인하고 4주 관찰부터. 단, 관찰에도 규율이 있다: **측정 항목은 그것이 판별할 결정과 짝지어져야만 등재된다.** 결정 없는 측정은 호기심이지 조사가 아니다(4주는 짧고 1인 서점의 측정 여력은 더 짧다).

| 관찰 항목 | 측정 방법 (1인 운영 가능 수준) | 이 데이터가 갈라놓을 결정 |
|---|---|---|
| O1. 유입량: 시간대·요일별 방문자 수 | 카운터 수기 체크(시간대별 정자 표기) | 문제가 "유입"인가 "전환"인가 — 1년 전략의 최상위 분기 |
| O2. 전환: 방문→구매 비율, 객단가 | POS 건수 ÷ O1 | 유입 마케팅 vs 매장 내 개선, 어디에 돈을 쓸 것인가 |
| O3. 상품 회계: 카테고리별 매출·마진·재고 회전, 6개월+ 무판매 재고 비율 | POS + 재고 대장 | 어떤 매대를 줄이고 무엇에 특화할 것인가 / 죽은 재고의 현금화 여부 |
| O4. 이탈 관찰: 들었다 놓은 책, 검색 후 빈손 퇴장, 짧은 대화("혹시 ~ 있어요?"의 기록) | 관찰 노트 + 문의 대장 | 구색의 문제인가 가격의 문제인가 경험의 문제인가 |
| O5. 고객 구성: 신규 vs 재방문(안면·적립 기준 추정), 근처 유동 인구의 시간 패턴 | 적립/수기 추정 | 전략의 대상이 "동네 단골 심화"인가 "외부 유입 창출"인가 |

**관찰 기간의 통제:** 4주간 가격·진열·운영시간을 동결(베이스라인 오염 방지). 단 청결·응대 같은 무해 개선은 예외. **계절성 한계의 명시:** 4주는 연중 대표성이 없다 — 그래서 이 플랜은 "4주 관찰로 끝"이 아니라 분기마다 경량 재관찰을 루프에 내장한다(아래).

## 문제 정의 / 목표

동네 서점의 1년 생존. 목표는 "좋은 전략의 실행"이 아니라 **한정된 현금과 체력을 관찰이 지목한 병목에만 쓰는 것.** 소상공인의 사인은 대개 나쁜 전략이 아니라 진단 없는 처방의 누적이다.

## 전략 = 관찰의 분기 (if-then, 우선순위는 돈의 크기)

**최상위 분기 (O1×O2):**
- **if 유입 적음 ∧ 전환 정상** → 병목은 존재 인지. 개입: 밖에서 보이는 것의 개선(간판·쇼윈도·지도 등록·지역 커뮤니티 노출). 유입 개선은 검증까지 느리므로 재관찰 주기를 8주로.
- **if 유입 정상 ∧ 전환 낮음** → 병목은 매장 안. O4와 교차: 구색 문제(찾는 책 없음 빈발)면 O3의 특화 결정으로, 배회 후 빈손이면 큐레이션·동선 개입으로. **유입 마케팅 지출 전면 금지** — 전환 안 되는 가게에 유입을 부으면 "안 사는 경험"만 대량 생산된다(부정적 구전의 확대재생산).
- **if 둘 다 낮음** → 자원상 동시 공략 불가. 전환부터(위 논거 — 유입은 전환이 준비된 뒤에).
- **if 둘 다 정상인데 적자** → 문제는 흐름이 아니라 구조(O3): 마진 구성 또는 고정비. 상품 회계 분기로.

**상품 회계 분기 (O3):**
- if 특정 카테고리가 마진·회전 동반 우위 → 그 방향으로 매대·매입 재배분(특화의 방향은 취향이 아니라 데이터가 정한다). 죽은 재고 비율이 높으면 → 반품/할인 현금화로 매입 자금 확보가 선행.
- if 책 마진만으로 고정비 불가 성립(관찰로 확정되면) → 이것은 개선 분기가 아니라 **모델 분기**: 공간의 이중 사용(대관·모임·구독 서비스)을 검토하되, 이 결정도 O5(고객 구성)가 근거 — 단골 기반이면 멤버십/모임형, 유동 기반이면 커피/체류형.

**고객 구성 분기 (O5):**
- if 재방문 중심 → 심화 전략(입고 알림, 취향 기록, 예약 주문) — 마케팅비 거의 0인 개입부터.
- if 뜨내기 중심 → 위치 자산 활용(창가·외부 노출) + 재방문 전환 장치(적립·다음 방문 쿠폰)를 먼저 실험.

**루프 규칙:** 한 번에 한 분기의 개입만 실행 → 8주 후 같은 지표 재측정 → 효과 판정(개입 전 대비) → 다음 병목으로. 동시 다발 개입은 금지 — 무엇이 들었는지 알 수 없게 되고, 1인 체력이 분산된다.

## 명시적 가정 목록 (틀렸을 경우의 영향 포함)

- **가정 1:** 4주+분기 재관찰로 계절성을 근사할 수 있다. → 틀리면(계절 진폭이 지배적) 분기 판정이 계절 노이즈에 오염 — 전년 동월 POS가 있으면 보정, 없으면 판정 보수화(큰 차이만 신호로).
- **가정 2:** 1인 운영자가 수기 관찰을 4주 지속 가능하다. → 틀리면 O1·O4를 표본일(주 3일)로 축소 — 정밀도보다 지속가능성(관찰이 중단되면 전부 무효).
- **가정 3:** 생존의 시간 여유가 최소 1년이다(현금 런웨이). → 틀리면(6개월 미만) 관찰 4주도 사치 — 이 플랜 전체가 즉시 현금화 분기(O3의 재고 정리, 고정비 협상)로 교체돼야 한다. **관찰 우선 프레임은 런웨이가 관찰 비용을 감당할 때만 유효하다.**

## 접근 및 단계

1. **0주: 관찰 설계 확정** — 위 표의 "갈라놓을 결정" 칸이 비면 항목 삭제. 측정 양식 준비(수기 가능 수준).
2. **1~4주: 베이스라인 관찰** (동결 유지).
3. **5주: 분기 판정** — 어느 분기인지 데이터로 확정, 첫 개입 1개 선택(판정 근거 기록).
4. **6~13주: 개입 실행 → 재측정 → 판정.** 이후 8주 주기로 루프.
5. **분기마다: 경량 재관찰(O1·O2만 1주)** — 계절 보정 + 새 병목 탐지.
6. **1년 시점: 루프 기록 전체가 곧 이 가게의 경영 데이터 자산** — 어떤 개입이 몇 %를 움직였는지의 개인화된 실측사(史).

## 대안과 기각 사유

- **베스트 프랙티스 즉시 이식(북토크·굿즈·SNS 등 일괄):** 기각. 병목 미상 상태의 다발 처방 — 무엇이 들었는지 영원히 모르고, 체력·현금만 소진. 이 프레임이 존재하는 이유가 이 대안의 실패율이다.
- **직관 기반 특화("내가 좋아하는 SF로 특화하자"):** 기각(보류). 취향은 O3 데이터와 일치할 때만 채택 — 일치하면 최상(지속 동력), 불일치하면 데이터 우선.
- **관찰 없이 모델 전환(카페 병설 등 큰 투자):** 기각. 가장 비가역적인 결정을 가장 정보 없는 시점에 내리는 것 — 모델 분기는 O3·O5의 판정을 통과한 뒤에만.

## 리스크 & 완화

- **R1: 관찰 4주의 기회비용 조바심.** → 관찰 자체가 개입의 일부임을 명시(문의 대장은 이미 고객 응대 개선이다) + 무해 개선은 병행 허용.
- **R2: 데이터가 애매(분기 판별 실패).** → 판정 보수화: 명확한 신호가 없으면 "비용 0에 가까운 개입"(진열 변경, 문의 도서 입고)만 실행하고 재관찰 — 애매할 때 큰 돈을 쓰지 않는 것 자체가 이 프레임의 배당금.
- **R3: 루프 피로(측정을 접게 됨).** → 재관찰을 경량화(O1·O2만) + 측정을 마감 루틴에 5분으로 내장.

## "완료"의 판정 기준 (플랜 문서로서)

- 모든 관찰 항목에 "갈라놓을 결정"이 짝지어져 있다(고아 측정 0).
- 전략 서술이 전부 if-then이며, 무조건 실행 항목은 존재하지 않는다.
- 개입은 1회 1개 + 재측정 + 판정 기록의 루프로 구조화돼 있다.
- 가정 3(런웨이)이 점검돼, 이 프레임 자체의 적용 가능 조건이 확인됐다.
