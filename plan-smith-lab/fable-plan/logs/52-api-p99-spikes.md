# 로그 — 52 API p99 스파이크 조사 (hypothesis-loop) [Fable]

## 진입 시 사고
이 프레임에서 가장 먼저 확정해야 하는 것은 가설이 아니라 **가설이 기각되는 방식**이라고 봤다. 그래서 첫 실질 섹션을 프레임 자체의 실패 모드 심문(조기 수렴, 가설 없는 계측 살포)으로 열고, 각 실패 모드에 대응하는 장치 — 기각 조건의 데이터 열람 전 확정, 선별 통과 전 가설당 1인일 상한, 신규 계측의 "가르는 가설 지명" 의무 — 를 먼저 박았다. 사실/추측 분리에서는 지시가 준 다섯 가설("GC다" 포함)을 전부 원장의 동등한 행으로 강등하는 문장을 의도적으로 넣었다. 레이턴시 조사의 관성적 첫수가 GC 튜닝이라는 걸 알았기 때문에, 그 관성을 대안 섹션에서 "부활 조건 없음"으로 명시 기각했다.

## 프레임이 만든 결정적 순간
1. **층위 이동 — 조사 단위를 지표에서 요청 코퍼스로.** "p99는 요청의 속성이 아니라 집계 창의 속성이다"라는 문장이 이 플랜의 축이다. 다섯 가설의 그림자가 대시보드 층위에서는 전부 "p99가 오른다"로 동일하다는 관찰이, 지문이 갈라지는 층위(요청 단위 + 차원 태그)로 내려가게 강제했다. 부수적으로 준스파이크 요청이 표본이 되면서 가정 1(재발 빈도)의 압박이 줄었다 — 프레임의 요구(기각 조건 등록)를 충족하려다 보니 표본 기아 문제까지 같이 풀린 경우다.
2. **좌표계 의심의 실체화.** 프레임의 required component를 축 목록(UTC↔로컬, 인스턴스, 호스트, 테넌트, 배포 에포크)과 **스파이크 간 간격 분포**로 구체화했고, 간격 주기성 발견 시 스케줄 작업 가설 H6으로 "승격"하는 경로를 만들었다. 좌표 스윕이 관찰 의례가 아니라 원장에 행을 추가하는 생성 장치가 된 지점.
3. **전멸 규칙 — 프레임의 재귀 적용.** 전 가설 기각 시 "스파이크는 단일 현상이다"라는 원장 밖 전제를 심문하고 코퍼스를 군집화해 군집별로 원장을 재실행한다. 두 원인이 다중화되면 어느 가설도 80% 조건을 못 넘겨 전멸이 나는 게 정상이라는 논리 — 기각 조건을 사전 등록했기 때문에 전멸이라는 사건 자체가 정보가 된다.
4. **시간 상자 기본값의 조건부화.** 10일 게이트의 기본값(헤지 요청)을 "H4 기각 시에만 허용"으로 묶었다. 재시도 폭풍 위에 헤징을 얹으면 증폭기가 되기 때문 — 완화 조치조차 원장의 판정 결과에 종속시킨 것이 프레임 일관성의 마지막 조각이었다.

## 프레임 이탈 / 관성 회귀 지점
- frames.md의 required components(사전 등록 기각 조건, 실험 비용의 원장 1급 필드화, 좌표계 의심) 자체는 전부 충족했다. 다만 좌표계 의심을 "원장 대조 전 의무 **1회**"로 고정한 것은 축소 해석이다 — 코퍼스가 커진 조사 중반에 재스윕할 트리거가 없어, 초기 데이터에서 안 보이던 군집은 놓칠 수 있다.
- 기각 임계값(70%/80%/리프트 3배)은 데이터 없이 뽑은 수치다. R3에서 자인하고 개정 이력 의무를 달았지만, "의미는 고정, 수치는 초깃값"이라는 처리 자체가 수치 자의성 문제의 해소가 아니라 관리로의 전환임은 남는다.
- **재현 레시피(완료 기준 1)와 가정 4의 긴장을 다루지 않았다.** H3(이웃 소음)이 채택 후보가 되면 통제 환경(스테이징/카나리아)에서 "이웃"을 재현하는 것은 원리상 어렵다 — 가설별로 재현 무대의 실현 가능성이 다른데, 완료 기준은 가설 무차별로 단일하다. 플랜 안에서 이 충돌의 처리 경로가 없다.

## 방향성
"판정 기준을 데이터보다 먼저 확정하고, 판정 불능(전멸)조차 사전 정의된 다음 수를 갖게 하라." 완화·계측·헤징 같은 모든 부수 행위를 원장의 판정 상태에 종속시키는 구조가 이 플랜의 지문이다.

## 자가 평가
프레임 충실도: 상. 세 required component가 전부 장치 수준으로 구현됐고(선언이 아니라 게이트·의무·승격 규칙), 재귀 적용(전멸 규칙)과 기본값의 조건부화까지 프레임 논리에서 연역됐다. 흠: ① H3 채택 시 재현 레시피의 실현 가능성 공백 — 완료 기준이 가설별 비대칭을 무시한다. ② 좌표 스윕의 1회 고정으로 조사 중반의 재스윕 트리거 부재. ③ 임계 수치의 자의성은 이력 의무로 관리했을 뿐 근거를 만들지는 못했다.
