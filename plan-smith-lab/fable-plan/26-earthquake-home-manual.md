# 재난 상황(지진) 대비 가정용 행동 매뉴얼
- 추론 프레임: 체크리스트 환원주의
- 한 줄 요약: 판단 제거는 세 층에서 이뤄진다 — 조건은 감각(보이면/들리면/냄새나면)으로, 역할은 이름으로, 미분류 상황은 단일 기본 행동으로. 그리고 본 체크리스트의 실행 가능성은 전적으로 평시 체크리스트가 "컴파일"해 둔 것에 달려 있다.

## 프레임이 요구하는 시작점 — 판단 제거의 설계 원칙

공포 상태의 뇌는 판단·계산·조율을 못 한다. 따라서:
1. **조건은 감각 동사로만:** "위험하면"(판단) 금지 → "가스 냄새가 나면"(감각) 허용. 모든 if는 오감으로 판별 가능해야 한다.
2. **조율은 이름으로 대체:** "누가 아이를 데려올지" 같은 런타임 협의를 제거 — 역할을 평시에 이름으로 박는다(엄마=가스·현관, 아빠=아이·가방 식으로, 각 가정이 기입).
3. **미분류 상황의 기본 행동 1개:** 체크리스트는 모든 경우를 못 덮는다. 덮이지 않는 모든 순간의 답을 하나로 통일: **"머리를 보호하고, 흔들림이 멈출 때까지 이동하지 않는다."** 모르면 이것 — 예외 상황에서 검색할 목차가 아니라 돌아갈 기본값이 있어야 한다.
4. **본 체크리스트는 평시 준비의 실행일 뿐:** "비상가방을 챙긴다"는 항목은 가방이 이미 싸여 있고 위치가 몸에 익었을 때만 항목이다 — 아니면 그것은 항목이 아니라 불가능한 과제다. 평시 체크리스트가 컴파일 타임이고, 재난 체크리스트는 런타임이다.

## 문제 정의 / 목표

지진 시 가정(아파트 기준) 행동 매뉴얼. 목표: **읽는 문서가 아니라 몸이 기억한 절차의 백업** — 현관문 안쪽에 붙는 1장, 항목마다 판단 0.

## 본 체크리스트 (현관문 부착 1장)

### ⓐ 흔들리는 동안 — 어디에 있든
- [ ] 탁자 밑으로. 없으면 → 팔·가방·방석으로 머리를 감싸고 그 자리에 웅크린다.
- [ ] 이동 금지. 불 끄러, 아이 방으로, 밖으로 — 전부 금지. (흔들림 중 이동이 최다 부상 원인. 아이에게도 평시에 "제자리 웅크림"을 가르치는 이유 — 부모가 가려는 판단을 하지 않도록.)
- [ ] 창·유리장·냉장고 옆이면 → 세 걸음 이내로만 낮게 이동해 멀어진다.

### ⓑ 흔들림이 멈추면 — 순서대로 (역할 이름 박기: 아래 [ ]에 가족 이름 기입)
- [ ] 발밑 확인 후 신발/슬리퍼. (유리 파편)
- [ ] [____]: 가스 밸브 잠금 — 위치: [____]
- [ ] [____]: 현관문 열어 고정. (뒤틀리면 안 열린다 — 탈출로 확보가 소지품보다 먼저)
- [ ] [____]: 아이/노약자/반려동물 상태 확인, 신발 신김.
- [ ] 전기 차단은 **대피 결정 후** 나가면서. (두꺼비집 위치: [____])

### ⓒ 대피 여부 — 감각 조건문 (판단 금지, 관찰만)
- **다음 중 하나라도 해당되면 → 즉시 ⓓ로:** 가스 냄새 / 타는 냄새·연기 / 벽·기둥의 금 또는 문이 안 닫힘 / 물 새는 소리 / 방송·재난문자의 대피 지시.
- **아무것도 해당 없으면 → 실내 대기.** TV·라디오·재난문자 확인. 여진 대비 ⓐ 반복 준비. (붕괴 조짐 없는 건물 안이 이동 중인 길보다 안전하다 — 이 판단은 이미 내려져 있다. 현장에서 다시 하지 않는다.)
- 애매하면 → **실내 대기가 기본값.** (애매함은 판단 신호가 아니라 기본값 신호다.)

### ⓓ 대피 시
- [ ] [____]: 비상가방 — 위치: [____]
- [ ] 두꺼비집 내리고 → 계단으로. **엘리베이터 금지.**
- [ ] 가방으로 머리 가리고, 건물 외벽·간판·유리에서 떨어져 이동.
- [ ] 집결지: 1차 [____] / 1차 불가 시 2차 [____] — **찾지 말고 기다린다** (가족 수색이 2차 사고의 주원인. "집결지에서 만난다"가 약속의 전부).
- [ ] 연락: 전화 금지, 문자·재난 안전 앱만. (통화망 폭주 — 문자는 지연돼도 도달한다.)

## 평시 체크리스트 (본 체크리스트의 컴파일 — 이것 없이는 위 문서가 무효)

- [ ] 위 [____] 전부 기입 (역할 이름, 밸브·두꺼비집·가방 위치, 집결지 2곳).
- [ ] 비상가방 상비: 물, 손전등, 보조배터리, 호루라기, 상비약, 현금, 가족사진(이산 시 확인용), 두꺼운 양말. 반기 1회 점검(약 유효기간·배터리).
- [ ] 침대·소파 근처 가구 고정, 머리맡에 유리·무거운 것 두지 않기, 슬리퍼 침대 밑 상비.
- [ ] **연 2회 가족 훈련:** ⓐ 웅크림 10초 + ⓑ 역할 동선 걸어보기 + 집결지까지 실제로 가보기. 훈련한 체크리스트만이 재난에서 체크리스트다 — 안 한 것은 종이다.
- [ ] 아이 교육: "제자리 웅크림"과 집결지 이름 암기 (부모의 구조 판단을 제거하는 유일한 방법은 아이의 행동이 예측 가능해지는 것).

## 명시적 가정 목록 (틀렸을 경우의 영향 포함)

- **가정 1:** 주거가 내진 설계 이후의 아파트/공동주택이다. → 노후 단독·조적조라면 ⓒ의 기본값(실내 대기)이 위험할 수 있음 — 이 매뉴얼의 기본값은 주거 유형 확인 후에만 유효하며, 그 확인은 평시 체크리스트의 선행 항목이다.
- **가정 2:** 가족 구성원이 역할 수행 가능하다(성인 2인 가정 기준 서식). → 1인 가구면 역할란을 전부 본인으로, 순서만 유지. 영유아·거동 불편 가족은 역할 재배분 필수.
- **가정 3:** 재난문자·방송 수신 가능. → 통신 두절 시나리오에서는 ⓒ의 감각 조건문만으로 동작하도록 설계돼 있음(외부 정보는 조건의 하나일 뿐 필수 입력이 아님 — 의도된 설계).

## 접근 및 단계 (제작 절차)

1. 행동을 국면(ⓐ~ⓓ)으로 분해하고 각 항목을 감각 동사 기준으로 작성.
2. **판단어 감사(audit):** 초안의 모든 항목에서 "위험하면/가능하면/상황 봐서/신속히" 류를 색출해 감각 조건 또는 기본값으로 치환. "신속히"도 금지 — 속도 부사는 지시가 아니라 불안 증폭제다.
3. 조율 지점(둘이 동시에 해야 하는 일)을 색출해 역할 이름란으로 치환.
4. 1장 인쇄·코팅·현관문 부착 + 평시 체크리스트는 냉장고에 별도 부착(둘의 물리적 분리 — 런타임 문서와 컴파일 문서는 다른 곳에 산다).
5. 첫 가족 훈련에서 문서 결함(막히는 항목·헷갈리는 조건)을 수집해 개정 — 훈련은 검증이기도 하다.

## 대안과 기각 사유

- **상세 안내서(원리·배경 설명 포함 소책자):** 기각. 재난 중 독해 불가. 교육용으로는 가치 있으나 그것은 평시 훈련의 교재이지 현관문의 문서가 아니다 — 두 문서의 목적을 섞으면 둘 다 실패한다.
- **"상황에 따라 유연하게 대처" 지침:** 기각. 프레임이 금지하는 것의 정의 그 자체.
- **스마트폰 앱 의존:** 기각. 정전·통신 두절·폰 파손이 지진의 기본 조건이다. 종이+몸의 기억이 1차, 디지털은 보조.

## 리스크 & 완화

- **R1: 기본값(실내 대기)이 틀리는 예외 상황(가정 1 위반 주거).** → 평시 체크리스트에 주거 유형 확인을 선행 항목으로 — 기본값의 유효 조건을 컴파일 타임에 검증.
- **R2: 훈련 미실시로 문서가 종이로 남음.** → 훈련을 연 2회 달력 고정(양력 절기 등 기억 가능한 날) + 훈련이 15분을 넘지 않게 설계(부담이 실행의 적).
- **R3: 기입란 공란 방치.** → 공란이 있는 한 "매뉴얼 없음" 상태임을 문서 상단에 명시 — 반쯤 컴파일된 문서가 가장 위험하다(있다는 안심 + 작동 안 함).

## "완료"의 판정 기준

- 본 체크리스트의 전 항목이 감각 조건·이름·기본값만으로 구성 (판단어 감사 통과 — "위험하면/신속히/상황 봐서" 0개).
- 모든 기입란이 채워지고, 가족 전원이 자기 역할을 문서 없이 말할 수 있다.
- 훈련 1회 완료 + 훈련에서 발견된 결함이 반영된 개정판이 부착돼 있다.
- 미분류 상황의 기본 행동(머리 보호·이동 금지)을 가족 전원(아이 포함)이 즉답한다.
