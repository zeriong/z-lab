# 동네 서점의 생존 전략 1년 플랜
- 추론 프레임: 관찰-데이터 우선
- 한 줄 요약: 전략 수립 전에 "무엇을 어떻게 측정/관찰할 것인가" 4주 관찰 설계를 먼저 세우고, 이후 전략은 관찰 결과의 분기(if-then)로만 서술한다.

## 프레임이 요구하는 시작점 — 전략 금지, 관찰 설계부터

"이벤트를 하자, 카페를 붙이자" 같은 전략을 지금 쓰는 걸 금지한다. 우리는 아직 **이 서점이 왜 어려운지 데이터로 모른다.** 그러니 먼저 4주간 무엇을 관찰할지 설계하고, 전략은 그 관찰 결과에 조건부(if-then)로만 매단다.

## 4주 관찰 설계 (전략보다 먼저)

**관찰 질문 → 측정 방법:**
- **O1 — 누가 오는가?** 시간대별·요일별 방문자 수, 연령/유형(동네 주민/학생/직장인), 체류 시간. → 수기 카운트 + 간단 관찰 기록.
- **O2 — 무엇을 사는가/안 사는가?** 매출 상위/하위 카테고리, 방문 대비 구매 전환율, 객단가, 재방문율. → POS 데이터 + 방문 대비 구매 비율.
- **O3 — 왜 안 사고 나가는가?** 둘러보고 나가는 사람의 행동(무엇을 집었다 놨나), 온라인/대형서점으로 이탈하는 이유. → 관찰 + 간단 대화/설문.
- **O4 — 지금 돈은 어디서 나고 새는가?** 카테고리별 마진, 재고 회전율, 고정비 구조. → 회계·재고 데이터.

**관찰 원칙:** 4주간은 큰 변화를 주지 않는다(관찰의 baseline 오염 방지). 측정만.

## 문제 정의 / 목표

동네 서점의 1년 생존. 목표는 "좋은 전략 실행"이 아니라 **관찰이 가리키는 곳에 자원을 쓰는 것**. 데이터 없는 전략은 도박이다.

## 전략 = 관찰 결과의 분기 (if-then)

전략을 미리 단정하지 않고, O1~O4 결과에 따라 갈라지는 조건부로 서술한다.

**O1(방문자) 분기:**
- if 방문자 자체가 적다 → 문제는 "유입". 전략: 지역 노출·발견성(간판·SNS·지역 커뮤니티). 
- if 방문은 많은데 구매가 적다 → 문제는 "전환"(O2로 이동). 유입 마케팅은 낭비.

**O2(구매/전환) 분기:**
- if 전환율 낮음 + 특정 카테고리만 팔림 → 진열/큐레이션 문제. 전략: 안 팔리는 재고 정리, 강점 카테고리 강화.
- if 객단가 낮음 → 전략: 연관 추천·묶음·굿즈.
- if 재방문 낮음 → 전략: 단골 프로그램·이벤트(단, O3가 이유를 확인한 뒤).

**O3(이탈 이유) 분기:**
- if "가격" 때문 → 전략: 가격 경쟁이 아니라 대형서점이 못 주는 것(큐레이션·경험·커뮤니티)으로 축 이동.
- if "찾는 책이 없어서" → 전략: 재고/주문 개선 또는 특화(니치 장르 전문).

**O4(수익 구조) 분기:**
- if 특정 카테고리가 적자 견인 → 전략: 축소/철수.
- if 고정비(임대료)가 구조적 문제 → 전략: 공간 재활용(대관·부업)으로 고정비 상쇄.

## 명시적 가정 목록 (틀렸을 경우의 영향 포함)

- **가정 1:** 4주 관찰이 연중 패턴을 대표한다. → 틀리면(계절성 큼) baseline이 편향. 시즌 보정 필요.
- **가정 2:** 문제의 원인을 현재 모른다(그래서 관찰이 필요). → 이 겸손이 프레임의 전제. 이미 원인이 명확하면 관찰은 시간 낭비.
- **가정 3:** 관찰할 데이터를 실제로 수집 가능(POS·인력). → 틀리면 관찰 설계를 수기·표본으로 downgrade.

## 접근 및 단계

1. **4주 관찰(개입 없음):** O1~O4 데이터 수집. baseline 확보.
2. **분기 판정:** 데이터가 어느 분기를 가리키는지 확정. 여기서 비로소 전략이 결정됨.
3. **가장 큰 병목부터 개입:** 관찰이 지목한 1순위 문제(유입/전환/이탈/수익)에 자원 집중.
4. **개입 후 재측정:** 같은 지표를 다시 재, 개입 효과를 데이터로 확인(관찰→개입→재관찰 루프).
5. **분기별로 반복:** 1순위 해결 후 다음 병목으로.

## 대안과 기각 사유

- **바로 전략 실행(이벤트·카페·SNS):** 기각. 프레임 위반이자 흔한 실패 — 문제가 "유입"인지 "전환"인지 모르고 유입 마케팅에 돈을 쓰면 낭비. 관찰이 진단을 선행해야 처방이 맞는다.
- **직관으로 원인 단정:** 기각. 사장의 직관이 O3(진짜 이탈 이유)와 다를 때가 많음. 데이터가 직관을 검증/반박해야.

## 리스크 & 완화

- **R1: 관찰 4주가 "아무것도 안 하는" 조급함을 부름.** → 관찰도 행동임을 명시. 다만 저비용 개선(정리·청결)은 baseline 오염 안 되는 선에서 병행 허용.
- **R2: 데이터가 여러 분기를 동시에 가리킴.** → O4(수익 영향) 크기로 우선순위. 적자 견인 요인부터.

## "완료"의 판정 기준

- 전략이 **관찰 데이터(O1~O4)의 분기로 도출**됨(직관·유행 아님).
- 개입 후 **같은 지표 재측정으로 효과가 확인**됨(관찰→개입→재관찰 루프 성립).
- 자원이 **관찰이 지목한 최대 병목**에 우선 투입됨(만능 전략 나열 아님).
