# 재난 상황(지진) 대비 가정용 행동 매뉴얼
- 추론 프레임: 체크리스트 환원주의
- 한 줄 요약: 모든 내용을 "판단이 필요 없는" 체크리스트로 환원한다. 판단이 남으면 그 판단을 사전에 내려 조건문으로 바꾼다.

## 프레임이 요구하는 시작점 — 판단을 제거한다

지진 순간의 사람은 공포·혼란 상태다. 이때 "상황을 판단하라"는 지시는 실패한다. 그래서 이 매뉴얼의 원칙: **읽는 사람이 판단할 게 하나도 없어야 한다.** 판단이 필요한 곳은 미리 판단을 내려 `if 조건 → then 행동`으로 바꾼다. 설명·이유·배경은 전부 제거(현장에선 못 읽음).

## 문제 정의 / 목표

가정에서 지진 발생 시 행동 매뉴얼. 목표는 "정보 제공"이 아니라 **공포 상태에서도 판단 없이 실행 가능한 지시서**. 판단이 남아 있는 항목은 실패로 간주.

## 체크리스트 (판단 제거, 시간순)

### 흔들리는 중 (0~1분) — 즉시
- [ ] 튼튼한 탁자 아래로 들어간다. 없으면 → 방석/가방으로 머리를 가리고 웅크린다.
- [ ] 탁자 다리를 잡는다.
- [ ] 창문·유리·큰 가구에서 떨어진다.
- [ ] (요리 중이었다면) → 그대로 둔다. 불 끄러 가지 않는다. 움직이지 않는다.
- [ ] 밖으로 뛰어나가지 않는다. (흔들림 중 이동 = 위험)

### 흔들림이 멈춘 직후 (1~3분)
- [ ] 신발을 신는다(유리 파편).
- [ ] 가스 밸브를 잠근다. (위치: ____ — 미리 기입)
- [ ] 두꺼비집(누전차단기)을 내린다. (위치: ____ — 미리 기입)
- [ ] 현관문을 연다(문틀 뒤틀림으로 갇힘 방지).
- [ ] 가족 위치를 확인한다.

### 대피 여부 판단 → 사전 결정된 조건문
- if 가스 냄새가 난다 OR 건물에 균열/기울어짐이 보인다 OR 화재 → **즉시 대피.**
- if 위 신호가 없다 → **실내에서 대기**(여진 중 계단·외부가 더 위험할 수 있음). 라디오/재난문자 확인.
- (판단을 남기지 않기 위해: "위험해 보이면"이 아니라 "가스 냄새/균열/화재"라는 **관찰 가능한 신호**로 조건을 고정)

### 대피 시 (대피로 결정된 경우)
- [ ] 비상가방을 든다. (위치: ____ )
- [ ] 엘리베이터 사용 금지 → 계단으로.
- [ ] 낙하물 주의하며 머리 보호하고 이동.
- [ ] 대피 장소로 간다. (지정 장소: ____ — 미리 기입)

### 대피 후
- [ ] 가족 집결 지점에서 인원 확인. (지점: ____ )
- [ ] 재난 방송/문자로 정보 확인.
- [ ] 연락: 전화 폭주 시 → 문자/메신저 우선.

## 사전 준비 체크리스트 (평상시 — 이걸 해둬야 위가 작동)
- [ ] 가스 밸브·두꺼비집 위치를 매뉴얼에 기입해 둔다.
- [ ] 비상가방(물·손전등·상비약·호루라기·현금)을 정해진 곳에.
- [ ] 가족 집결 지점·대피 장소를 미리 합의.
- [ ] 큰 가구 고정(전도 방지).

## 명시적 가정 목록 (틀렸을 경우의 영향 포함)

- **가정 1:** 사용자는 공포 상태라 판단력이 급감한다. → 이 가정이 "판단 제거" 원칙의 근거. 맞다는 전제가 재난 매뉴얼의 정설.
- **가정 2:** 대피 여부는 "가스/균열/화재"라는 관찰 신호로 대부분 결정 가능. → 틀리면(애매한 중간 상황) 조건문이 커버 못 하는 케이스 발생 → 기본값을 "실내 대기"로 두어 안전 측 오류.
- **가정 3:** 사전 준비(위치 기입·비상가방)가 돼 있다. → 안 돼 있으면 현장 체크리스트가 공란을 만나 멈춤. 그래서 사전 준비를 별도 체크리스트로 강제.

## 접근 및 단계 (제작 관점)

1. **행동을 시간순 국면으로 분해:** 흔들리는 중 / 직후 / 대피판단 / 대피 / 대피후.
2. **각 항목에서 판단을 색출:** "위험하면", "상황 보고" 같은 판단어를 찾아 관찰 가능한 조건으로 치환.
3. **설명 제거:** 이유·배경을 삭제(현장에선 안 읽힘). 왜가 궁금하면 별도 교육 자료로 분리.
4. **공란(사전 기입 항목) 표시:** 집집마다 다른 정보(밸브 위치 등)는 빈칸으로 두고 사전 준비 체크리스트로 채우게.

## 대안과 기각 사유

- **정보 풍부한 설명형 안내서:** 기각. 프레임 위반. 공포 상태에서 긴 설명은 안 읽힘. "지진의 원리"는 매뉴얼이 아니라 교육 자료의 몫.
- **"상황에 맞게 판단하세요":** 기각. 판단을 사용자에게 남기는 것 자체가 실패. 판단은 제작 시점에 내려 조건문화.

## "완료"의 판정 기준

- 매뉴얼의 **모든 항목이 판단 없이 실행 가능**(판단어 0개, 남은 판단은 관찰 가능 조건문으로 치환됨).
- 애매한 상황의 **기본값이 안전 측**으로 정해져 있음(예: 애매하면 실내 대기).
- 집집마다 다른 정보는 **사전 준비 체크리스트의 공란**으로 분리돼, 현장 체크리스트가 멈추지 않음.
