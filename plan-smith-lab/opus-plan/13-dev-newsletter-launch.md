# 개발자 대상 유료 뉴스레터 런칭 계획
- 추론 프레임: 단위경제 우선
- 한 줄 요약: 구독자 1명의 LTV/CAC 모델을 먼저 세우고, 숫자가 성립하는 조건에서만 실행 계획을 전개한다. 성립 안 하면 피벗.

## 프레임이 요구하는 시작점 — 구독자 1명의 경제학

실행 아이디어(콘텐츠, 마케팅) 이전에, **구독자 한 명이 돈이 되는가**부터 계산한다. 이 숫자가 성립하지 않으면 아래 어떤 실행 계획도 무의미하다.

**LTV (구독자 1명의 생애가치)**
- 가격 가정: 월 $10.
- 이탈률 가정: 월 5% churn → 평균 구독 수명 = 1/0.05 = **20개월**.
- LTV(마진 전) = $10 × 20 = **$200**. 콘텐츠 원가(내 시간)를 변동비로 안 잡으면 총마진 ≈ LTV.

**CAC (구독자 1명 획득 비용)**
- 유료 광고 경로: 개발자 타깃 CPC ~$1~2, 방문→무료구독 전환 ~5%, 무료→유료 전환 ~3%.
- → 유료 1명당 방문 필요 ≈ 1 / (0.05 × 0.03) ≈ 667방문 × $1.5 ≈ **~$1,000 CAC**. **경보: LTV $200 « CAC $1,000. 유료 광고로는 붕괴.**
- 오가닉 경로(SNS/커뮤니티/추천): 금전 CAC ≈ $0, 대신 내 시간이 비용.

**판정: LTV/CAC = $200/$1,000 = 0.2 (유료광고). 성립 불가.** 3배 이상이어야 건강한데 0.2다. → **유료 광고 기반 실행 계획은 이 자리에서 기각.** 오가닉/추천 기반이어야만 숫자가 성립.

## 숫자가 성립하는 조건 도출

LTV/CAC ≥ 3을 만족시키려면:
- **경로 A — CAC를 0에 수렴(오가닉):** 광고 대신 무료 콘텐츠로 유입 → CAC(금전)≈0이면 LTV/CAC는 시간 대비로만 평가. 이 경로가 유일하게 성립.
- **경로 B — LTV를 키운다:** 가격↑($10→$30, 팀/기업 구독) 또는 churn↓(월 5%→2%면 수명 50개월, LTV $500). 그래도 $1,000 CAC엔 못 미침 → 광고는 여전히 불가, 단 오가닉 경로의 여유가 커짐.
- **결론: 경로 A(오가닉 유입) 필수 + 경로 B(가격/리텐션)로 LTV 보강.**

## 문제 정의 / 목표

개발자 대상 유료 뉴스레터를 지속 가능한 사업으로. 목표는 구독자 수가 아니라 **단위경제가 성립하는 성장 구조**.

## 명시적 가정 목록 (틀렸을 경우의 영향 포함)

- **가정 1:** 월 churn 5%. → 틀려서 10%면 수명 10개월·LTV $100로 반토막, 오가닉 경로마저 빡빡해짐. **가장 민감.**
- **가정 2:** 무료→유료 전환 3%. → 틀려서 1%면 오가닉 유입 요구량이 3배로 뛰어 시간 비용 폭증.
- **가정 3:** 가격 $10에 저항 없음. → 틀리면 LTV 전제 붕괴. 개발자 무료 대안(블로그)이 많아 지불의향이 핵심 미지수.

## 접근 및 단계 (숫자가 성립하는 경로만)

1. **무료 티어로 오가닉 유입(경로 A):** 무료 뉴스레터/글로 구독 리스트를 먼저 키운다. CAC(금전)≈0. 유료는 그 위에 얹음.
2. **전환 퍼널 실측:** 방문→무료→유료 전환율을 소규모로 측정해 가정 1·2를 실제 숫자로 대체. 이게 다음 결정의 근거.
3. **가격·리텐션 최적화(경로 B):** 연간 결제 할인(churn↓), 기업/팀 구독(가격↑, LTV↑)으로 LTV를 CAC 대비 안전 마진까지.
4. **유료 광고는 오직 LTV/CAC≥3이 실측으로 확인된 세그먼트에만** 켠다(기본은 오프).

## 대안과 기각 사유

- **유료 광고로 빠른 성장:** 기각. 도입부 계산에서 LTV/CAC=0.2. 돈을 태울수록 빨리 죽음. 단위경제 프레임의 존재 이유가 이 기각.
- **무료로만 운영(수익화 포기):** 기각(목표 위반). 다만 무료 티어는 유입 수단으로 유지.
- **저가 대량(월 $3):** 기각. LTV가 더 낮아져 오가닉 경로의 시간 대비 수익성이 악화. 개발자 대상엔 소수 고가치가 유리.

## 리스크 & 완화

- **R1: 오가닉 유입이 느려 성장이 정체.** → 오가닉은 원래 느림. churn을 낮춰 LTV를 키우는 쪽으로 무게 이동(경로 B). 조급하게 광고 켜지 않음.
- **R2: 가정(churn·전환)이 낙관적.** → 2단계 실측 전엔 확장 결정 보류. 숫자가 성립을 반증하면 즉시 피벗(가격·타깃·포맷).

## "완료"의 판정 기준

- **LTV/CAC ≥ 3**이 특정 채널에서 실측으로 확인됨(가정이 아니라 데이터).
- 성장이 **금전 CAC≈0인 오가닉/추천**에 기반함(광고 의존 아님).
- churn·전환·가격의 실측값이 초기 가정과 대조돼, 모델이 현실로 보정됨.
