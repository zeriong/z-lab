# 자취 독립 첫 이사 D-day 매뉴얼
- 추론 프레임: checklist
- 한 줄 요약: 이사 당일의 판단을 0으로 만든다 — 모든 결정은 사전에 내려 조건문으로 박아두고, 당일 문서에는 시간대별·역할별 실행 체크박스와 단 하나의 폴백만 남긴다.

## 프레임이 요구하는 시작점 — 판단을 작성(사전) 시점으로 당긴다

이 프레임의 규율은 "당일 문서에 판단이 한 줄이라도 남으면 실패"다. 당일의 나는 잠 못 잔 고스트레스 상태라 판단력이 평소의 절반이다. 그러므로 판단은 지금(사전) 전부 내리고, 당일에는 조건문 실행만 남긴다. 판단 제거를 세 층으로 강제한다.

- **(가) 감각 조건화.** 조건은 감각으로 판별 가능해야 한다. "짐이 위험하면 따로" 금지 → "액체·유리·칼은 빨간 테이프 붙은 박스에만". "냉장고 상태 확인" 금지 → "냉장고 뒤 물받이에 물이 보이면 수건으로 닦고 세워둠 표시".
- **(나) 조율 제거 = 사전 배정.** "누가 할지"를 당일에 협상하지 않는다. 모든 항목에 실행자 이름이 붙는다(본인 / 도우미 / 기사). "짐 어디 둘지 그때 정하자" 금지 → 방 배치도를 미리 그려 박스마다 방 이름 스티커.
- **(다) 미분류 = 단일 폴백.** 분류 안 되는 물건·상황은 그 자리에서 고민하지 않고, 미리 정한 단 하나의 행동으로 흘려보낸다(아래 별도 섹션).

여기에 두 개의 구조 규칙을 건다.

- **컴파일/런타임 분리.** 문서는 두 개다. 사전 준비 체크리스트(컴파일 타임)와 당일 체크리스트(런타임). **당일 체크리스트의 품질은 사전 체크리스트가 결정한다.** 사전 목록에 공란이 하나라도 있으면 당일 매뉴얼은 무효다 — 반쯤 준비된 문서는 없는 것보다 위험하기 때문이다.
- **금지어 감사.** 문서 전체에서 "적당히·알아서·상황 봐서·되도록·빨리·나중에·필요하면"을 색출해 수치나 감각 조건으로 치환한다. "빨리 출발" → "08:00 출발", "필요하면 도우미 호출" → 도우미는 D-1까지 확정.

## 문제 정의 / 목표

생애 첫 자취 독립 이사를, 당일에 어떤 즉흥 판단도 없이 실행 가능한 매뉴얼로 만든다. 목표는 "짐을 옮기는 것"이 아니라 **"판단 0으로 옮기는 것"**이다. 짐 이동은 기사가 하고, 매뉴얼이 지키는 것은 판단이 무너지기 쉬운 지점(정산·행정·안전·첫 밤)이다.

## 명시적 가정 목록 (틀렸을 경우의 영향)

- **가정 1 (플랜 전체가 걸림): 이사 당일은 고스트레스·저인지 상황이다.** → 틀리면(경험 많고 짐 적은 저스트레스 이사) 이 프레임 자체가 과잉이다. checklist는 고스트레스·저판단 상황 전용이며, 여유로운 상황에 적용하면 오적용이다.
- 가정 2: 일반 이사(반포장, 기사 2~3인)이며 도우미 1인(친구·가족)을 확보한다. → 틀리면(혼자 이사) 역할 축이 붕괴 → 시간축만 남기고 폴백을 본인 단일로 재편해야 함.
- 가정 3: 옛집·새집 모두 엘리베이터/사다리차·주차가 사전 예약 가능하다. → 틀리면 상차·하차 시간 블록이 통제 불가 노드가 되어 버퍼 조건문을 추가해야 함.
- 가정 4: 인터넷·공과금·전입신고는 당일이 아니라 사전/사후에 처리 가능하다. → 틀리면 당일 블록에 행정 항목이 끼어들어 판단 0 원칙이 흔들림.

## 사전 준비 체크리스트 (컴파일 타임) — 공란 0이 발효 조건

각 항목은 D-1 밤 22:00까지 체크되어야 하며, 공란이 1개라도 있으면 당일 매뉴얼은 무효(실행 금지)다.

- [ ] 이사비 잔금 결제 수단·금액 확정(현금/계좌 명시). 실행자: 본인
- [ ] 옛집 관리비·도시가스·전기 정산 예약 완료, 정산 담당자 연락처 시트에 기입
- [ ] 보증금 반환 계좌·일정 집주인과 문자로 확정(구두 금지)
- [ ] 인터넷 이전/설치 일자 예약(당일 아닌 날로)
- [ ] 전입신고·확정일자 방문/온라인 일정 예약(사후 처리)
- [ ] 방 배치도 1장 작성 → 새집 방마다 이름(거실/침실/주방) 부여
- [ ] 모든 박스에 방 이름 스티커 + 내용물 한 줄. 액체·유리·칼 박스에 빨간 테이프
- [ ] 귀중품 가방 구성: 현금·신분증·계약서·열쇠·충전기·상비약 (트럭에 절대 안 실음)
- [ ] 첫 밤 생존 박스 1개 별도: 침구·세면도구·수건·화장지·물·폰 충전기
- [ ] 역할·연락처 시트: 본인 / 도우미 / 이사업체 대표번호 / 옛집·새집 관리사무소
- [ ] 사다리차·엘리베이터·주차 예약 확인(옛집·새집 양쪽)
- [ ] 우천 대비: 박스 덮을 큰 비닐/방수포 준비

공란 발생 시 폴백: 그 항목을 (a) 도우미에게 위임하거나, (b) "당일 스킵 가능"으로 사전 표시할 수 없다면 이사 일정을 미룬다. 무효 규칙은 협상 대상이 아니다.

## 당일 체크리스트 (런타임) — 시간대별 × 역할별

각 블록은 "완료 판별식"(감각으로 확인 가능한 종료 조건)을 갖는다.

**07:00 상차 전 (본인)** — 두꺼비집/가스 밸브 위치 확인, 귀중품 가방을 몸에 착용. 판별식: 귀중품 가방을 멘 채로 다음 블록 진입.

**기사 도착 ~ 출발 (기사=운반 / 본인=검수 / 도우미=문·엘베 고정)** — 빨간 테이프 박스는 본인이 직접 지목해 세워 싣게 함. 판별식: 방 배치도의 방 개수 = 실린 박스 그룹 수, 미분류 박스는 마지막에 싣는다.

**이동 (본인=귀중품·열쇠 / 도우미=옛집 최종 점검)** — 도우미는 옛집 빈 상태를 사진 촬영(각 방·베란다·냉장고 뒤). 판별식: 방 수만큼 사진이 찍혔는가. 본인은 옛집 열쇠·카드키 반납 또는 지정 장소 배치.

**새집 도착 ~ 기사 떠나기 전 (기사=하차 / 본인=하자 사진 / 도우미=박스 방 배분)** — 하자(벽·바닥·수도·문) 사진을 먼저 찍고, 박스는 스티커대로 각 방에 내려놓는다. 판별식: 거실 중앙에 남은 것은 미분류 박스뿐.

**기사 떠난 직후 (본인)** — 이사비 잔금 결제, 영수증/이체 캡처 저장. 판별식: 결제 캡처 1장이 폰에 있는가.

**22:00 첫 밤 (본인)** — 첫 밤 생존 박스만 연다. 판별식: 미분류 박스를 뒤지지 않고 잘 곳·씻을 것·마실 물·폰 충전 4개가 확보되는가.

## 미분류 폴백 — 단 하나의 행동

- 물건이 분류 안 됨 → 미분류 박스에 넣고 새집 거실 중앙에 모은다. 그 자리에서 판단하지 않는다. 판단은 이사 다음 날로 미룬다.
- 상황이 매뉴얼에 없음 → 결정권자(본인) 1인에게 귀속. 본인이 즉시 판단 불가면 그 일은 멈추고 다음 체크박스로 넘어간다. 절대 즉흥 판단하지 않는다. "판단이 필요한데 못 하겠는 일"은 실패가 아니라 다음 날로 이월되는 항목이다.

## 대안과 기각 사유 (부활 조건)

- 당일에 융통성 있게 판단하기(반사적 정답). 기각: 고스트레스에서 융통성은 곧 실수. 부활 조건: 이사 경험 3회 이상 + 짐 소량 + 도우미 불필요한 저스트레스 이사.
- 포장이사 전면 위임(체크리스트 불필요). 기각: 첫 자취 예산 초과 가정 + 위임해도 정산·행정·첫 밤은 본인 몫이라 판단 0 문제가 안 사라짐. 부활 조건: 예산 확보 시 물리 운반만 위임하고 정산·행정·첫밤 체크리스트는 유지.
- 시간순 단일 리스트(역할 구분 없음). 기각: 조율 제거 실패 — 당일에 "누가 할지" 협상이 재발. 부활 조건: 혼자 이사면 역할 축이 없으므로 시간축 단일로 회귀.

## 리스크 & 완화

- R1: 사전 준비 시간 부족으로 공란 발생 → 매뉴얼 무효. 완화: 사전 목록을 D-7에 시작, 항목별 마감일 부여. 공란 폴백(위임/스킵표시/연기)을 미리 규정.
- R2: 미분류 폴백이 본인에게 몰려 병목. 완화: 물건 폴백은 "판단 미루기"(사람 개입 0), 사람 판단이 필요한 폴백만 1차로 도우미에게.
- R3: 통제 불가 노드(기사 지각·엘베 고장·비). 완화: 조건문화 — "기사 30분 이상 지각 시 도우미가 대표번호로 전화, 본인은 검수 계속", 우천 시 방수포는 사전 항목.
- R4: 프레임 오적용 — 판단이 정말 필요한 항목(하자 심각도, 계약 분쟁)까지 체크박스화. 완화: 그런 항목은 "전문 판단 필요"로 분리해 당일이 아닌 사전/사후 시간대로 이동. 당일 문서에는 "사진만 찍고 다음 날 판단"으로 축약.

## "완료"의 판정 기준

- 사전 체크리스트의 모든 박스가 D-1 밤 22:00에 체크됨(공란 0). 공란이 1개라도 있었다면 무효 규칙이 발동해 당일 실행을 막았는가 — 실측.
- 당일 어느 시점에도 "이거 어떻게 하지?"로 5초 이상 멈춰 즉흥 판단한 횟수 = 0. 멈춤이 발생했다면 그것은 사전 체크리스트의 결함으로 기록된다.
- 귀중품(현금·신분증·계약서·열쇠)이 이사 트럭 상차 목록에 없고 24시간 내내 본인 가방에 있었다 — 실측.
- 새집 첫 밤 22:00에 미분류 박스를 열지 않고 잘 곳·씻을 것·마실 물·폰 충전 4개가 확보되었다 — 실측.
- 문서 전체에서 금지어("적당히·알아서·되도록·빨리·나중에·필요하면") 검색 결과 = 0건.
